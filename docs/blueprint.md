# **App Name**: Citizen Simplified

## Core Features:

- PDF Upload and Preview: Allow users to upload PDF documents and view a preview.
- AI Document Simplification: Use AI to convert complex documents into simplified summaries.
- Structured Output Generation: Generate structured, easy-to-read summaries with key sections, powered by generateObject.
- Conversational Chat Interface: Enable users to ask follow-up questions and receive answers via a chat interface, potentially powered by a tool for including the uploaded PDF when relevant.
- Multi-Language Support: Support multiple languages, particularly Hindi, for document simplification and chat responses.
- Document Export: Export the simplified explanation as a clean, shareable PDF.
- User Authentication and History: Allow users to create accounts, save their document history, and access saved chats.

## Style Guidelines:

- Primary color: Deep blue (#1A535C) to convey trust and authority.
- Background color: Very light blue (#E8F5F9) for a clean, readable layout.
- Accent color: Warm orange (#F26419) to highlight key actions and information.
- Body text font: 'PT Sans' for a clear, readable, and modern style.
- Headline Font: 'Space Grotesk', sans-serif, for titles and headings.
- Use simple, clear icons from lucide-react to represent key actions and information. Use plain language and emojis for extra clarity.
- Subtle animations using Framer Motion for loading states and transitions to enhance the user experience.
